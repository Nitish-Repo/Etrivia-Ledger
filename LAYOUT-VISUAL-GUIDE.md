**Key Concepts:**

1. **One Layout, Multiple Pages** - Layout wraps all private pages
2. **Dual Navigation** - Menu AND Tabs both navigate
3. **Synchronized State** - Route changes update both menu & tabs
4. **Gesture-Enhanced** - Swipe adds extra navigation option
5. **Lazy Loading** - Pages load only when needed
